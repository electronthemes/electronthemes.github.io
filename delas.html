## Setup custom pages with templates You can set custom page using page template like `tag list`, `author lists`, `portfolio` ### Tag List Page Create a page with any name and select `Tags` custom page template from page settings ![](/assets/images/common/tag-list.png)

### Author List Create a page name `Authors` or anything and select the page template `Authors` from the settings. ![](/assets/images/common/author-list.png)


### Portfolio List Create a page name `Portfolio` or anything and select the page template `Portfolio` from the settings. ![](/assets/images/common/portfolio-list.png) To show the post as portfolio in the portfolio page, provide `portfolio` tag in the post. ![](/assets/images/common/portfolio.png) ### Popular news Section To add posts on popular news section, add `popular` tag on the post. The post will come automatically in the popular news section. ![](/assets/images/common/popular.png) ### Setup posts to show per page ```js "config": { "posts_per_page": 7 } ``` Change `posts_per_page`’s value as your wish on `package.json` file in the theme file. After changing this you need to `restart` your ghost ```bash ghost restart ```
